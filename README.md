<p>Project is already hosted on firebase

Link: <a href='https://weather-app-d2a09.web.app/'>https://weather-app-d2a09.web.app</a>

<h2>Development</h2>
<p>If you have some ideas, you can add them to this project, just make some steps:</p>
<ol>
<li>install nodejs (if you don't have)</li>
<li>clone project, by using git</li>
<li>open project with editor</li>
<li>Before running open terminal and install dependencies:
<br />
<code>cd <<i>project-name</i>></code>
<br />
<code>npm install</code>
</li>
<li>Before running this app, you should create account in <a href='https://openweathermap.org/'>https://openweathermap.org</a>. You should use api-key, which is located in your account. In root of cloned project create .env file and enter this line:
<br />
<code>REACT_APP_API_KEY=<<i>api-key</i>></code>
<br />
<li>running project on localhost:
<br />
  <code>npm start</code>
</li>
</ol>
